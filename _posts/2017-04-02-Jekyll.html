记录一下 Jekyll 环境在 ubuntu14.04 下的安装。

前言： 由于 jekyll 是由 Ruby 编写，第一步自然是安装 Ruby. 网上许多方法都是直接 apt-get install ruby，这样安装的是1.9.3版本，Jekyll 3.x 需要 Ruby>=2.0 以上才能运行， 这点在 jekyll 官网有说明http://jekyllrb.com/docs/installation/（中文官网没有说明这一点）, 而 ubuntu 直接使用 apt-get 安装 ruby2.0 的话，需要运行ruby2.0而不是ruby，的确可以通过 ln -sl 链接修改默认， 但是这样后续安装 jekyll 还是会报错，也不推介使用apt安装jekyll; 这里我使用 rvm 安装 ruby; 跟 nvm 安装nodejs一样道理。

1.安装 rvm {% highlight ruby %} $ curl -L https://get.rvm.io | bash -s stable {% endhighlight %} 安装好后需要 $ source ~/.rvm/scripts/rvm 每次开机都需要连接一下。

2.修改rvm使用国内镜像源，速度会提高很多. {% highlight ruby %} $ echo "ruby_url=https://cache.ruby-china.org/pub/ruby" > ~/.rvm/user/db {% endhighlight %} 参考RVM官网：http://ruby-china.org/wiki/rvm-guide
使用 rvm list known 查看可安装的 ruby 版本
eg：
# MRI Rubies
[ruby-]1.8.6[-p420]
[ruby-]1.8.7[-head] # security released on head
[ruby-]1.9.1[-p431]
[ruby-]1.9.2[-p330]
[ruby-]1.9.3[-p551]
[ruby-]2.0.0[-p648]
[ruby-]2.1[.10]
[ruby-]2.2[.6]
[ruby-]2.3[.3]
[ruby-]2.4[.0]
ruby-head
使用 $ rvm install 版本号 安装ruby
修改默认的使用的ruby版本
$ rvm 版本号 --default
查看ruby版本
{% highlight ruby %} $ ruby -v >>ruby 2.2.0p0 (2014-12-25 revision 49005) [x86_64-linux] {% endhighlight %} 查看gem版本
{% highlight ruby %} $ gem -v {% endhighlight %} 接下来就可以按照jekyll官网说的使用gem安装jekyll。
